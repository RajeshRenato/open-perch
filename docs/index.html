Installation Guide

Up and running
in under a minute

Platform-specific setup — because first-launch quirks shouldn't slow you down.

1

Download the .dmg file — choose Apple Silicon (M1/M2/M3/M4) or Intel based on your Mac

2

Open the .dmg and drag Perch into your Applications folder

3

Launch Perch — macOS will block it because it's not from the App Store. That's normal.

4

Go to System Settings → Privacy & Security, scroll down to the Security section, and click Open Anyway

5

Click Open in the confirmation dialog — you only need to do this once. Perch will appear in your menu bar.

Shortcut: Press Cmd+Shift+R anywhere to open the popup. You can customize this key in the app settings.
1

Download Perch Setup 1.1.0.exe from the releases page

2

Run the installer — Windows Defender SmartScreen will show a warning (the app isn't code-signed yet)

3

Click More info on the SmartScreen popup, then click Run anyway

4

Follow the installation wizard. Perch will start automatically and appear in your system tray (bottom-right)

Shortcut: Press Ctrl+Shift+R anywhere to open the popup. If Windows Defender flags it later, add Perch to your antivirus exceptions.
1

AppImage — download, make executable, and run:

chmod +x Perch-1.1.0-x86_64.AppImage
./Perch-1.1.0-x86_64.AppImage
2

Debian/Ubuntu (.deb) — install with dpkg:

sudo dpkg -i Perch-1.1.0-amd64.deb
3

Perch will appear in your system tray. Click the icon to open the popup.

Shortcut troubleshooting: The global shortcut Ctrl+Shift+R may conflict with other apps (especially terminals). If it doesn't work:
• Check Settings → Keyboard → Shortcuts on GNOME for conflicts
• On Wayland, Perch falls back to X11 mode — try clicking the tray icon if the shortcut doesn't respond
• You can change the shortcut key inside the app: open Perch main window → Settings
