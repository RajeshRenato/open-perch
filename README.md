# Perch

A minimal clipboard snippet manager that perches in your menu bar.

## Download

Download the latest release for your platform from the [Releases](https://github.com/RajeshRenato/open-perch/releases) page.

| Platform | Download |
|----------|----------|
| macOS (Apple Silicon) | `Perch-x.x.x-arm64.dmg` |
| macOS (Intel) | `Perch-x.x.x.dmg` |
| Windows | `Perch Setup x.x.x.exe` |
| Linux | `Perch-x.x.x-x86_64.AppImage` or `.deb` |

## Features

- Clipboard History - Automatically captures text and images
- Saved Snippets - Pin frequently used text
- Global Shortcut - `Cmd/Ctrl+Shift+R`
- Quick Search - Find any snippet instantly
- Cross-Platform - macOS, Windows, Linux
